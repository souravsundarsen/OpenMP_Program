Method 1:
=========================================================
To Compile an OpenMP Program
--------------------------------------
gcc -fopenmp omp1.c

To Run an OpenMP Program
--------------------------------------
./a

Method 2:
=========================================================
Go To Terminal Menu in Visual Studio --> Configure Default Build Task --> Select C/C++ compiler (A file named "tasks.json will open") 
--> go to line number 10 and replace "-g" with "-fopenmp"

"This setup for one time"

Now to Run
---------------------------------------
./program_name_without_extension
